Activate Geneious License

Follow these steps to activate your Geneious license using your Tufts email.

Note that users can activate Geneious on up to 2 devices.

Update to latest version (optional)

If you are still running v2023 or earlier, you will need to update Geneious first. If you are running v2024 onwards, you can skip this step.


Activate license

Go to Help > Activate License....


Email sign in

If the license activation window does not appear, your version of Geneious does not support the new license. Please update to the latest version of Geneious before proceeding.

Enter your email

Tufts email needs to be in the format of your firstname.lastname@tufts.edu.

Log in with SSO

Click on the Log in with SSO button to be redirected to the Tufts login page.

Enter password

Enter your Tufts password to log in.

All set!

Congratulations! Your Geneious license has been activated. It's worth mentioning that with the new license system, Tufts VPN is not required anymore to run Geneious!
